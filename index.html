Database instellen

Eenmalige koppeling met Supabase — gratis, geen creditcard nodig

1

Maak een gratis Supabase account

Ga naar supabase.com → "Start your project" → maak een account met Google of e-mail. Gratis tier is voldoende voor 20+ klanten.

2

Maak een nieuw project aan

Kies "New project" → geef het project een naam → regio: West EU (Ireland) → stel een wachtwoord in → klik "Create new project" (duurt 1 minuut).

3

Maak de tabel aan

In Supabase → ga naar SQL Editor → plak onderstaande code → klik "Run":

CREATE TABLE maandcijfers (
  id uuid DEFAULT gen_random_uuid() PRIMARY KEY,
  bedrijf text NOT NULL,
  sector text,
  periode text NOT NULL,
  email text,
  omzet numeric,
  brutomarge numeric,
  ebitda numeric,
  netto numeric,
  debiteurendagen numeric,
  kas numeric,
  solvabiliteit numeric,
  personeelskosten numeric,
  signaal text,
  created_at timestamptz DEFAULT now()
);

ALTER TABLE maandcijfers ENABLE ROW LEVEL SECURITY;
CREATE POLICY "Publiek lezen en schrijven" ON maandcijfers FOR ALL USING (true) WITH CHECK (true);
