Jaarcijfers aanleveren

📊 Exact Online
  1. Ga naar Rapportage → Financieel
  2. Kies Verlies- en Winstrekening
  3. Selecteer het boekjaar
  4. Klik op Exporteren → Excel
  5. Herhaal voor de Balans
📊 AFAS
  1. Ga naar Financieel → Rapporten
  2. Kies Resultatenrekening
  3. Selecteer het gewenste jaar
  4. Klik op Afdrukken → PDF
  5. Herhaal voor de Balans
📊 Snelstart
  1. Ga naar Overzichten → Resultatenrekening
  2. Kies het boekjaar en klik Tonen
  3. Klik rechtsboven op Exporteren
  4. Kies Excel (.xlsx)
  5. Herhaal via Overzichten → Balans
📊 Twinfield
  1. Ga naar Rapporten → Financieel
  2. Selecteer Winst- en verliesrekening
  3. Kies het jaar en klik Uitvoeren
  4. Export via Excel-icoon rechtsboven
  5. Herhaal voor Balansoverzicht
📊 King / Visma
  1. Ga naar Balans & Resultaat
  2. Kies Resultatenrekening + jaar
  3. Klik op Afdrukken → PDF
  4. Herhaal voor de Balans
📊 Accountant aanlevering
  1. Vraag uw accountant om de jaarrekening
  2. Gevraagd: V&W en balans als PDF
  3. Controleer dat het volledige jaar erin staat
  4. Upload beide bestanden hieronder
Tip: Weet u niet hoe? Stuur de bestanden gewoon op — wij lezen ze uit ongeacht het formaat. Bij twijfel: bel 06 4215 2499.
📂 Welke bestanden heeft u nodig?
Verlies- & Winstrekening (V&W)
Overzicht van uw omzet, kosten en winst over het jaar. In uw boekhoudprogramma ook wel "resultatenrekening" of "winst- en verliesrekening" genoemd.
Balans
Overzicht van uw bezittingen, schulden en eigen vermogen op een peildatum. Laat zien wat uw bedrijf heeft en wat het verschuldigd is.
Exporteer beide uit uw boekhoudprogramma (Exact, AFAS, Snelstart, Twinfield, e.a.) als PDF of Excel.
Verlies- & Winstrekening (V&W) — laatste jaar *
Balans — laatste jaar *
V&W — jaar daarvoor (aanbevolen voor trendanalyse)
Balans — jaar daarvoor (aanbevolen voor trendanalyse)
V&W — twee jaar geleden (optioneel)
Balans — twee jaar geleden (optioneel)

🛡 Backend beveiliging (Supabase)
Voor volledige beveiliging moeten de volgende maatregelen in Supabase worden ingesteld. Zonder deze instellingen kunnen klanten elkaars data ophalen via de API.
Stap 1 — Row Level Security aanzetten
ALTER TABLE uploads ENABLE ROW LEVEL SECURITY;
ALTER TABLE klanten ENABLE ROW LEVEL SECURITY;
ALTER TABLE rapporten ENABLE ROW LEVEL SECURITY;
Stap 2 — Policy: klant ziet alleen eigen data
CREATE POLICY "eigen data" ON uploads
  FOR SELECT USING (bedrijf = current_user);
Stap 3 — API-route beveiliging (Cloudflare Pages)
Voeg in uw functions/api/analyse.js een verificatie toe van het klant-token voordat data wordt teruggegeven. Neem contact op met Groen van Texel voor implementatiehulp.
Reset — schone lei

Verwijder alle klanten inclusief alle uploads, rapporten, basisscans en subsidiescans. Gebruik dit om een volledige testcyclus opnieuw te doorlopen.
